选择脑电图独立成分进行矫正的实用指南

0. Summary

本文综述了\(ICA\)分离出的五种典型的脑电伪影(眨眼、眼跳、肌肉噪声、不良通道和罕见事件)，以及目前自动化方法提供的几种检测这些伪影的方法。

1. Research Objective(s)

脑电信号中独立成分的选择；

2. Background / Problem Statement

2.1 方法

用 ICA 分离和减去独立的成分；这种分量减法被广泛用于从EEG记录中去除诸如眨眼或肌肉活动之类的伪影

2.2 常见伪影特征的特征说明；

  1. 眼睛成分

    特点：

    1. 最容易识别
    2. 地形基本上平坦
    3. 排在前十几成分中

    眨眼

    水平眼动

    误认眼动

  2. 肌肉成分

    ​ 来自颈部、颌部和面部肌肉的强直性肌肉活动在电极帽边缘的电极上产生一种固定的活动。虽然受试者通常被要求坐着不动放松，但由于打哈欠或吞咽等原因导致的无法控制的姿势活动以及肌肉活动可能会发生，并被记录在脑电图中。

  3. 坏通道

  1. 罕见事件

    B：事件发生在单个电极上

    C: 事件发生在多个电极上

  1. 模棱两可的成分

有的成分不能明确区分，反映信号的混合，应小心处理。见1-3最后图像。

2.4 其他插件

  1. FETHER(全自动统计阈值脑电伪影剔除）
  2. ADJUST(基于时空特征联合使用的脑电伪迹自动检测）
  3. MARA(多重伪像去除算法)
  4. CORRMAP：专门设计用于通过使用IC地图和用户选择的模板地图之间的相关程度来检测眼睛和心脏伪影的工具

3. Method(s)

3.1 数据集预处理程序：

  1. 目测原始信号以排除数据的不良部分，
  2. 如表2所述的重新参考、下采样和滤波，
  3. 历元，
  4. 前刺激基线去除，
  5. 使用扩展Infomax算法(来自EEGLAB)的ICA。

3.2 任务和测量

  • Tools>Select Components by MAP

  • 检查每个组件以决定是否应该拒绝它

  • 丢弃给定组件的“理由”：

    “眨眼”、“扫视”、“肌肉”、“孤立通道”、“少数试验”(用于识别罕见事件)或“其他”中的任何一个。

  • 三个专家们独立对所有训练数据集进行评级后，一起修改评级，直到达成共识。无法达成共识的组成部分被归类为“其他”。

  • 两位专家(包括作者MC)使用相同的程序检查了13个测试数据集的920个组成部分，并一致将其分类

4. Evaluation

5. Conclusion

到目前为止，ICA并不是唯一可用于校正伪迹的方法。许多其他方法允许在丢弃伪迹的同时潜在地保留神经信号。这些方法包括低通滤波、回归方法、主成分分析(Ille等人，2002；McMenamin等人，2009；Wallstrom等人，2004)、信号空间投影(Uusitalo和Ilmoniemi，1997)或典型相关分析(Clercq等人，2006)。但是ICA是一种强大的方法，它允许同时校正几种类型的伪迹。它用于去除EEG中的伪影，也用于去除脑磁图(MEG)数据中的伪影(Barbati等人，2004年；Parra等人，2005年)，以及伴随的功能性磁共振成像在EEG中产生的强伪影(Mantini等人，2007年)。

事实证明，它在矫正眼部(Jung等人，2000b)、心脏(Campos Viola等人，2009年)、肌肉(Creso-Garcia等人，2008年；McMenamin等人，2010年；Olbrich等人，2011年)或发作性伪影(Urrestarazu等人，2004年)方面是有用的。

ICA的也有缺陷。因此，如何正确地对伪迹进行分类是很重要的。

6. Notes

  1. EEG信号伪迹去除时发生的 I型和II型错误
    • I 丢失待检测的伪迹
    • II 错误分类非伪迹成分
