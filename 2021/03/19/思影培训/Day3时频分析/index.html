时域vs频域

  • 时域分析： 脑电波幅随时间的变化（如 刺激后 - 刺激前）。 ERP
  • 频域分析：分析脑电信号各频段的频谱能量。
    • 主要适用场景：静息态数据等。
    • 对各被试、各通道、各分段的脑电信号做傅里叶变换，得到各个频率点上的能量，再做统计分析。
    • 统计方式：
      • （1）沿着频率点统计：选择感兴趣的空间区域（通道），沿着频率点进行统计检验。
      • （2）沿着通道统计：选择感兴趣的频段，沿着通道进行统计检验。
      • （3）提取特定通道、特定频段的能量数值，进行统计检验。
  • 时频分析
    • 既有时域信息 又有频域信息

参数检验（正态分布效果好）：

T检验：

  1. 单样本T检验： 一组被试数据 和 某个数值作比较 ttest

  2. 配对T检验： 一组被试两种不同条件的数据作比较（刺激1 vs 刺激2；干预前 vs 干预后） ttest

  3. 双样本T检验： 两个独立分组的被试的数据作比较 （实验组 vs 对照组） ttest2

方差分析

F检验

  1. 单因素方差分析： 多个(>=2) 独立分组的被试的数据的比较 （实验组1 vs 实验组2 vs 对照组） anova1

  2. 单因素重复测量方差分析： 一组被试多种 (>=2) 不同条件的数据的比较 （刺激1 vs 刺激2 vs 刺激3；前测 vs 中测 vs 后测）anova_rm

  3. 两因素方差分析： （疾病：患者、对照； 性别：男、女 问题： 患者相对于对照的变化，是否会随着性别的改变而改变？）anova2

  4. 两因素混合方差分析：（干预方式：干预组、对照组； 时间：前测、后测。 问题：后测相对于前测的差异 会不会 随着干预方式的改变而改变？） mixed_anova

单个被试、单个通道时频变换结果的汇总

%% 单个被试、单个通道时频变换结果的汇总
 
%导入一例被试的数据
filename = 'D:\MyWorkSpace\Matlab\SiYingPeiXun\25EEG_day3\Example_data\1_LH.set'; 
EEG = pop_loadset(filename);

%提取某个通道的数据
%  x： 时间点*分段        （EEG.data  通道*时间点*分段）
x = squeeze(EEG.data(29,:,:)); 

%分段时间范围
xtimes=EEG.times/1000;  
%时频结果的  时间分辨率
t=EEG.times/1000;
%时频结果的  频域分辨率
f=1:1:30;  
%数据采样率
Fs = EEG.srate;
%窗长   （如果分段时间较短，窗长可以适当缩短；
%在段的起始、结束处，时频变换的结果不是绝对精准的，因为做了补0操作）
winsize = 0.200; 
%时频变换 （短时傅里叶变换）
%P ， 频率 *  时间 * 分段  （Power）
[S, P, F, U] = sub_stft(x, xtimes, t, f, Fs, winsize); 
%P_data  分段平均后的结果   频率*时间
P_data=squeeze(mean(P,3)); 

%对时频结果，减基线（凸显ERD/ERS）  
% 刺激后能量-刺激前能量
%选择基线范围，刺激前0.8 ~0.2毫秒，避免了窗长问题
t_pre_idx=find((t>=-0.8)&(t<=-0.2));

% 对于每一个频率点
for i=1:size(P_data,1)
    %提取该频率点所有时刻的power
    temp_data=squeeze(P_data(i,:));
    %该频率点所有时刻的power  减去   基线时间段的power的均值
    P_BC(i,:)=temp_data-mean(temp_data(t_pre_idx)); 
    %P_BC，基线校正后的结果，频率*时间点
end

figure;  
%绘制 未做基线校正的时频图
subplot(211); imagesc(t,f,P_data); axis xy; 
hold on;  axis xy; colorbar;
xlabel('Time (ms)','fontsize',12); ylabel('Frequency (Hz)','fontsize',12); 
title('TFR (without baseline correction)','fontsize',15);
%绘制基线校正后的时频图
subplot(212); imagesc(t,f,P_BC); axis xy; 
hold on;  axis xy; colorbar; %caxis([-0.8 2]);
xlabel('Time (ms)','fontsize',12); ylabel('Frequency (Hz)','fontsize',12); 
title('Baseline-corrected TFR','fontsize',15);

%% ttest for each time-frequency point

% P_data: 被试 * 条件 * 通道 * 频率 * 时间点
% 提取所有被试、所有条件、13号通道（空间ROI、Cz）、所有频率、所有时间的power
%: data_test： 被试 *  条件 * 频率 *  时间点
data_test=squeeze(P_data(:,:,13,:,:)); %% select the data at Cz, data_test: subj*cond*frequency*time
%对于每一个频率点
for i=1:size(data_test,3)
    %对于每一个时间点
    for j=1:size(data_test,4)
        % 挑选第一种条件的数据
        data_1=squeeze(data_test(:,3,i,j)); %% select condition L3 for each time-frequency point
        %挑选第二种条件的数据
        data_2=squeeze(data_test(:,4,i,j)); %% select condition L4 for each time-frequency point
        % 配对T检验
        [h,p,ci,stats]=ttest(data_1,data_2); %% ttest comparison
        % 存储第i个频率点、第j个时间点的p值
        P_ttest(i,j)=p; %% save the p value from ttest
        %  存储T值
        T_ttest(i,j)=stats.tstat; %% save the t value from ttest
    end
end


%对于每一个时频点
for i=1:size(data_test,3)
    for j=1:size(data_test,4)
        % 挑选所有条件的数据
        data_anova=squeeze(data_test(:,:,i,j)); %% select the data at time-frequency point
        %重复测量方差分析
        [p, table] = anova_rm(data_anova,'off');  %% perform repeated measures ANOVA
        %存储各时频点的p值
        P_anova(i,j)=p(1); %% save the data from ANOVA
        % 存储F值
         F_anova(i,j)=table{2,5}; %% F value from ANOVA
    end
end
