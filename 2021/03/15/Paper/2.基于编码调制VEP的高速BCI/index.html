A high-speed BCI based on code modulation VEP

发表于2021-03-15|更新于2021-05-11|论文
|字数总计:685|阅读时长:2分钟|阅读量:

0. Summary

目标数提升到32个，提高了ITR，

多通道处理可以提高准确率

1. Research Objective(s)

提高ITR和识别准确率

2. Background / Problem Statement

BCI 通信系统传输速率低，提高ITR(信息传输率)

3. Method(s)

  • 提出了基于c-VEP的调制系统，其中二进制伪随机码被用来调制不同的视觉刺激
  • 提出了一种提高识别精度的多通道检测方法

4. Evaluation

  1. 该系统由一个脑电图放大器和一台带CRT显示器的个人电脑（PC）组成。

  2. 不同目标的之间的圆移关系是目标识别的基础，一旦获得了T0的模板，其他目标的模板可以很容易地获得

    目标设置符合等效邻域原则

  3. ​ 圆移

​ (a) Ts 为刺激周期 （\(T_s = 63/60)=1.05s\))

​ (b) T0 编码

​ (c)T0 相应信号

​ (d) T1编码(T0延时\(\tau _s\)) \(\tau _s = 2/60=0.033s\)

​ (e) T1响应信号（延时）

​ (f)T1模板可有T0（圆移获得）

  1. 在获得所有目标的模板后，可以采用模板匹配的方法进行目标识别，通过选择相关系数最大的目标来确定固定目标.

    • 选取T20作为参考，记录N个周期数据\(x_n(t),n=1,1,...N\)

    • N个周期取平均获得模板 \[ M_{20}(t)=\frac{1}{N}\sum_{n=1}^Nx_n(t) \]

    • 循环移位获得所有模板 \[ M_k(t)=M_{20}(t-(\tau_k-\tau_{20})) , k=0,1,...31 \]

    • 计算相关系数 \[ \rho_k=\frac{<M_k(t),x(t)>}{\sqrt{<M_k(t),M_k(t)><x(t),x(t)>}} \]

    • 相关系数最大的即为目标

  2. 多通道处理可提高识别精度

    训练阶段：

    1. 记录k个周期多通道数据\(X\in R^{C\times (k\times L)}\)

    2. 对k段X平均得到多通道响应R

    3. 复制R k次，得到\(S\in R^{C\times (k \times L)}\)

    4. 用CCA计算是X和S相关系数最大的现象变换\(W_x,W_s\)

    测试阶段：

    ​ 用\(W_x\)作为空间滤波系数进行在线处理数据

​

5. Conclusion

双极法与多通道法准确率比较

16目标与32目标 BCI 性能比较

本文对高速c-VEP BCI系统的基本原理和实现方法进行了详细的介绍,提出的c-VEP BCI具有108±12.0比特/分钟-1的高ITR,超过了以往基于脑电图的BCI的记录。

改进：

  2. target编码
