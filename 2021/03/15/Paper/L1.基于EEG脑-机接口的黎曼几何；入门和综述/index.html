Riemannian geometry for EEG-based brain computer interfaces ;a primer and a review

发表于2021-03-15|更新于2021-06-04|论文
|字数总计:1.4k|阅读时长:4分钟|阅读量:

0. 总结

黎曼几何在脑机接口的应用，简述了黎曼分类的原理，优缺点。

1. 研究目的

对黎曼几何在脑-机接口中的应用进行了综述，并对基于黎曼几何的分类框架进行了初步探讨。

提高脑-机接口的稳健性。

2. 背景与问题陈述

实现所需的可用性和健壮性的途径不是通过提高系统的复杂性，而是“我们需要在技术进步和现实世界的实际使用之间取得平衡”。将预处理、特征提取和分类不作为独立的过程，而是作为一个整体。

3.1 黎曼分类框架入门

对称正定(SPD)矩阵(以下称为正矩阵)可以被认为是多维平方实数的推广。特别地，一维随机变量的方差是正数(平方和)，而N维随机变量的方差是正矩阵，通常称为协方差矩阵。

为实现到均值的最小距离(MDM)，需要一个合适的距离函数(度量)和相应的均值函数。

度量(距离):

  • 非负的
  • 只有当两个元素相等时它才等于零
  • 对称的
  • 服从三角不等式。

度量空间（S，d)。S代表空间，d代表距离；

常用欧拉距离为 \[ d_E(a,b)=|a-b| \] 相应的黎曼均值即算术均值为计算

\[ m=\frac{1}{K}\sum_kc_k \] 实际用对数变换的均值，集合均值 \[ d_G(a,b)=|loga-logb|=|log{\frac{a}{b}}| \] 特点：

  • 尺度不变性 \(d_G(xa,xb)=d_G(a,b)\), 所有的\(a,b,x>0\)
  • 反演不变性 \(d_G(a^{-1},b^{-1})=d_G(a,b)\) ,所有的\(a,b,x>0\)

变换后的均值为毕达哥拉斯均值

\[ g=\sqrt[K]{c_1.c_2...c_K }=exp(\frac{1}{K}\sum_klogc_k) \] 算术均值与集合均值的比较

由于对称性，此矩阵有三(\((N(N+1)/2)\))个变量。

由柯西不等式

3.2 优点

  1. Sensor空间和源空间之间的等价性
  2. 集合均值的鲁棒性
  3. 泛化能力

3.3 与CSP比较

通用空间模式（CSP）算法是一种特征提取方法，它使用空间滤波器来最大化两个类的可区分性。

可归结为特征值-特征向量分解

3. Method(s)

传统解码器：预处理，特征提取，分类

模式：MI，ERP，SSEP

目前都有专门的预处理，信号处理和分类模块

两大范式 优点 缺点
hard machine learning 泛化性能好 需要大量数据，计算密集型
利用信号处理提高信噪比，然后分类（空间滤波等） 快速训练，成本低 泛化差

黎曼分类：

MDM(Minimum Distance to Mean)：到平均数的最小距离

MDM 算法是最简单的黎曼方法

3.1高级黎曼分类器

切线空间映射是一种局部投影，它将流形的元素映射到欧几里得空间，保持它们的距离关系完整。

一旦投影到切线空间，数据点（即正矩阵）可以被向量化以形成标准特征向量，从而馈入任何标准分类器（例如线性判别分析、逻辑回归、支持向量机等）。

虽然基于切线空间映射的黎曼方法的整体性能优于 MDM 方法，并且它们明显优于现有技术 [102]，但由于算法复杂性增加和可能需要大量学习，因此它们不太适合在线操作

3.2 黎曼几何应用于脑电图的研究综述

黎曼几何在 EEG 领域的使用已被独立引入两组，用于分类睡眠阶段 和基于运动想象的 BCI

4. Evaluation

LDA 线性判别分析

5. Conclusion

  1. 提供了一个更简单的黎曼分类方法，
  2. 最小距离到平均数（MDM）的初步介绍，我们主要依靠直观的（几何）解释。Riemannian MDM方法完全基于两个简单的概念：两个数据点之间的距离和其中若干点的平均值。
  3. 黎曼几何学提供了处理对称正定矩阵的天然框架，许多种类的结构协方差矩阵都是这种类型。不管协方差矩阵是如何定义的，MDM黎曼分类器对于所有三种BCI模式，即运动想象、事件相关电位和稳态诱发电位都是一样的。
  4. 基于切线空间映射的黎曼方法的总体性能优于MDM方法，而且性能明显优于最新的技术水平，但由于分类器继承了算法复杂度的增加和可能需要高强度的学习，它们不太适合在线操作
  5. 空间滤波器的改进只能为分类目的带来适度的改进，而且这种改进并不容易转化为可靠性和鲁棒性的显著提高。
