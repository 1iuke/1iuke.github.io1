2. 研究背景及问题陈述

2.1 BCI研究

  1. 提出了一种新的频率-相位联合调制方法，用0.5s的闪烁信号标记40个目标，并开发了一种使用单个校准数据的特定用户目标识别算法
  2. Edelman等人。分别和同时使用具有感觉运动节律的二维光标控制任务和具有稳态视觉诱发电位(SSVEP)的视觉注意任务来研究认知灵活性的概念。他们发现，单独和联合执行这两项任务的精确度没有显著差异
  3. Ortner等人的研究成果。应用异步SSVEP脑机接口进行矫形器控制。6名未经训练的受试者表现出良好的控制性，阳性预测值(PPV)在60%以上，所有受试者的总体PPV高达78%[5]。
  4. Pfurtscheller等人的研究成果。进行了基于脑机接口的混合式矫形器的自定步操作的研究，其中有无基于图像的大脑开关，仅在需要控制时才激活基于SSVEP的四步矫形器，并在休息期间停用SSVEP脑机接口。这两种心理策略的结合显示，与单独使用SSVEP脑机接口相比，静息期的假阳性率要低得多。
  5. 孟等人。研究了使用非侵入性脑-机接口控制机械臂来完成抓取任务，发现受试者可以高精度地控制机械臂来执行需要多个自由度的任务[7]。
  6. Xu等人。提出了一种创新的基于微型事件相关电位的脑-机接口范式，它使用了一种空码分多址(SCDMA)方案来减少视觉疲劳[8]。

2.2 VEP

基于VEP的BCI具有用户训练少和ITR高的特点

根据调制方法可分为：

  • t-VEP 时间调制
  • f-VEP 频率调制
  • c-VEP 编码调制
  • s-VEP 空间调制

其中f-VEP和c-VEP有更高的ITR，应用潜力大，但是c-VEP相对于f-VEP研究还不够深入

2.3 c-VEP

现有c-VEP范例将所有刺激目标打包为一组，用原始伪随机二进制码及其循环移位码 或 相同长度的不同伪随机二进制码调制。前者只需训练一个目标来创建所有目标的模板。后者需要创建所有目标的模板。通常采用第一种以减少训练时间。

可利用的目标数目收调制码长的限制。例如，如果两个相邻刺激之间的时延为2比特，则63比特的M序列可以循环右移31次，并且最多提供32个刺激。

另一方面，影响ITR的主要因素之一是可选择目标的数量。高性能的脑机接口通常有大量的目标。因此，如何增加目标数目是c-VEP脑机接口的关键问题。

3. 研究方法

在[另一篇论文](\2021\04\15\Paper\A Novel c-VEP BCI Paradigm for Increasing the Number of Stimulus T argets Based on Grouping Modulation With Different Codes)的研究中，通过将现有的第一种目标调制方法与第二种基于视觉刺激分组调制的方法相结合，构建了一个48个目标的c-VEP BCI。在不显著增加训练时间的情况下，大幅增加刺激目标的数量和由此产生的ITR。

在本研究中，将目标群体的数量从三个扩展到四个，将目标排列的方法从等效邻居原则调整为分开放置，并应用了四种不同的数据预处理方法来分析c-VEP系统。每个目标组有16个刺激目标，共64个目标。对8个被试的实验数据的分类结果验证了该范式在增加刺激靶数和ITR方面的可行性和有效性。

3 Statistical analysis

针对系统性能指标非正态分布的特点，采用Wilcoxon符号秩检验对中值差值显著性进行统计分析，研究了不同目标数和不同数据预处理方法对系统性能的影响。配对测试是在95%的置信水平下进行的。

5. 结论

1 Auto- and cross-correlation of c-VEP signals

虽然c-VEP BCI中的视觉刺激作为二进制信号都是不相关的，但这并不能保证对这些刺激的反应也是不相关的，因为大脑是一个非线性动态系统。因此，刺激目标的可分性实际上取决于刺激响应(即，c-VEP信号)的正交性，而不是编码

![](A multi-target brain-computer interface based on code modulated visual evoked potentials/image-20210415170258948.png)

四个自相关函数中的每一个在时滞0处都有一个尖锐的中心峰值，在其他地方都有一个低旁瓣。中心峰值与最大旁瓣的比值足够大，可以保证在同一组中准确区分不同的目标。

![](A multi-target brain-computer interface based on code modulated visual evoked potentials/image-20210415170351280.png)

互相关函数在所有时间滞后时都具有较小的值。这意味着也很容易区分不同组中的目标。

2 Amplitude spectrum and SNR

频谱分析是为了找出c-VEP信号的主要功率集中在哪个频段。主要能量基本上位于2-40 Hz的频率范围内。因此，在时间上对频率中的原始c-VEP信号进行滤波以降低噪声提高信噪比

![](A multi-target brain-computer interface based on code modulated visual evoked potentials/image-20210415171557406.png)

为了评估c-VEP信号的信噪比，探索了四种不同的数据预处理方法，即：

1)M1：未经时间滤波的原始单通道(Oz)信号；

2)M2：带时间滤波的单通道(Oz)信号；

3)M3：未经空间滤波的原始多通道信号；

4)M4：同时进行时间和空间滤波的多通道信号。

M4为c-VEP脑机接口系统中使用的方法。使用来自每个参考目标的训练信号来评估SMI，并且SNR是从四个参考目标导出的四个SMI的平均值。

![](A multi-target brain-computer interface based on code modulated visual evoked potentials/image-20210415172029255.png)

(M4)时间滤波和空间滤波组合的信噪比改善效果最好。

6. 笔记

ITR由三个参数决定，即分类精度、目标数量和检测时间。高ITR要求第一个参数高，第二个参数大，第三个参数短。然而，这三个参数并不是独立的，而是相互作用的，这使得脑-机接口系统的设计成为一项复杂的任务。

高的ITR主要归因于以下几个因素：

1)较多的刺激目标，

2)具有良好的自相关和互相关特性的四种不同的调制码，

3)结合时间和空间滤波的有效信号处理算法

4)用于目标识别的短数据段。

改进：

在本系统中，64个刺激目标被呈现在尺寸为24英寸(1920×1080像素)的面积受限的LCD监视器上，使得每个刺激目标的大小为140×100像素，这对于VEP BCI来说是次优的。以前的研究建议，对于尺寸为1920×1080像素的显示器，最佳刺激尺寸至少为170×170像素(视角为3.8˚)。因此，增加每个刺激目标的大小将提高检测精度和由此产生的ITR。考虑到个体的差异，选择特定于受试者的参数，例如通道位置、滤波器频带和训练试验的次数，对于提高个体的表现也是有用的。

用于目标识别的数据长度是一个刺激周期，即对于60 Hz的屏幕刷新率为1.066秒。使用固定停止(FS)策略的缺点是它没有考虑被试在BCI容量上的个体差异。最好使用不同的数据长度对目标进行分类，即采用动态停止(DS)策略，该长度由脑电特征的质量和每次试验的分类置信度确定。DS策略不仅可以提高分类精度，而且可以缩短目标识别的整体时间。最近的一项研究表明，即使对于高速SSVEP BCI，DS策略也可以进一步提高它们的性能[39]。因此，将DS策略纳入64个目标的c-VEP BCI将是未来研究的一个方向
